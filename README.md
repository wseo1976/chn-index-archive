# chn-index-archive

## 项目简介

chn-index-archive 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何单个域名或具体网站，仅作为页面内容的存储与展示空间。所有归档页面均为独立 HTML 文件，可直接通过浏览器访问。

## 目录说明

```
├── index.html          # 仓库主页，列出所有归档页面入口
├── pages/              # 存放归档 HTML 页面文件的目录
│   └── ...             # 具体页面文件
└── README.md           # 本文件
```

- `index.html` — 仓库首页，提供所有归档页面的索引链接。
- `pages/` — 所有归档的 HTML 页面文件均放置于此目录下，便于统一管理和引用。

## 页面归档说明

- 每个归档页面均为独立的 HTML 文件，不含外部依赖或跨域资源。
- 归档页面内容可能来源于不同时期、不同来源，仅作保存与展示用途。
- 页面文件名通常反映其内容或归档时间，便于识别。
- 本仓库不保证归档页面的时效性或完整性，亦不对页面内容做额外解释或评价。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接打开 `index.html` 查看归档页面列表。
3. 点击列表中的链接，或直接在浏览器中打开 `pages/` 目录下对应的 HTML 文件。
